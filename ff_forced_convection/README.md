This folder contains the files used to investigate the differences between using a uniform or axially varying friction factor in forced convection. 

In detail, this folder contains:
- _deltaP_evaluation.ipynb_: notebook comparing the two options;
- *_aux.py*: module containing auxiliary variables and functions;
- _simulation_ subfolder: contains simulations' input and outputs;
- _base_input_ subfolder: contains base input and base power ditributions to be modified.
